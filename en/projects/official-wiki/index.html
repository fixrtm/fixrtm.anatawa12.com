fixRTM Official Wiki

The wiki which has information about RTM and fixRTM.

Description

This is the official wiki of fixRTM. This wiki will have many information for fixRTM as officially and for RTM and KaizPatchX unofficially.

This wiki uses wiki.js, a simple multi-locale supported wiki system based on markdown and other text platforms.

The wiki.js doesn't have enough features for me so fixRTM uses fork of wiki.js. The fork has following features:

  • The fork has Accept-Language Header based redirection to translated pages.
  • The fork has translated login pages.
  • The fork has link to terms of service on login page.
  • The fork disallows to create page in non-markdown syntax.
