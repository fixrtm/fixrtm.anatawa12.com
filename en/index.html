GOALS

The goal of fixRTM is "Improve user experience of RTM". To achive this GOAL, There's several small goals below. Each goals are explained below.

  • Improves RTM
  • Provides tools for model pack authors

Improves RTM

RTM has several bugs but original author doesn't have enough time to fix all them. That's why this project will fix bugs of RTM. RTM also have some missing features which many needs them. This project will implement them.

Subprojects for this goal

  • fixRTM
  • Fork of KaizPatchX
  • Plugin Permissions for NGT

Provides tools for model pack authors

Currently, there's no tools and documentation for modelpacks of latest RTM. This project is planned to provide documentation and linter.
