jar-in-jar-mod Gradle Plugin

modのjarのサイズを小さくするための Gradle Plugin

Description

これは fixRTM の配布サイズを小さくするための Gradle Plugin です。 このプラグインにより fixRTM のサイズがだいたい半減しました。

Download

ダウンロードはありません。 Gradle の plugins ブロックを使用してください。 プラグインのリストは GitHub 上の README を参照してください。
