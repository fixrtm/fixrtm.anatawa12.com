Mod fixRTM

1.12.2 版 RTM と NGTLib にパッチを当てるMOD

Description

fixRTMの主なサブプロジェクトです。 このModは RTM のバグを修正し、いくつかの機能を追加するためのmodです。 詳細は GitHub のプロジェクトを参照してください。

Download

  • stableかSNAPSHOT リリースを CurseForge から
  • 最新版リリースを Github から
  • すべてのリリースを Github から

dependencies

  • fixRTM リリース時の RTM の最新版
  • fixRTM リリース時の NGTlib の最新版
  • Minecraft Forge 14.23.5.2847 以降
こちらに RTM と NGTlib と fixRTM の間のバージョンの対応表があります。 各 fixRTM のリリースのリリースノートと mod のメインとなるクラスの @Mod アノテーションに前提バージョンの情報があります。
