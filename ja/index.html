Community

GOALS

fixRTM のゴールは「RTMのユーザー体験を良くする」ことです。 そのため、以下のゴールを定めました。

  • RTM を改善する
  • ModelPack 作成のためのツールを作成する

RTM を改善する

RTM はいくつかバグがありますが原作者は修正するのに十分な時間がありません。 そのためこのプロジェクトでは RTM のバグを修正します。 多くの人が望むが RTM に実装されていない機能もあるため、それらも実装します。

  • Plugin Permissions for NGT

Provides tools for model pack authors

現在、最新の RTM 向けのモデルパックのためのツールやドキュメントは見つかりません。 そのためこのプロジェクトではツールやlinterを作成します。
